This is a browser game implemented in ActionScript3. This game was meant to be a small-scope project to test small team dynamics and see if we could produce a finished product. I built the team, wrote portions of the gameplay code, managed source control, and set things up in Kongregate to "ship" the game. Ultimately it went over schedule and some things didn't make it into the game, but it was a good learning experience.
